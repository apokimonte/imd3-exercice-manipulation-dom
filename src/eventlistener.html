Exercices - Ecouteurs d'événement

Rappel théorique

Les événements en JavaScript

Les événements permettent de détecter et réagir aux interactions de l'utilisateur (clics, saisie clavier, mouvements de souris, etc.) ou aux changements dans le navigateur.

La méthode addEventListener()

addEventListener() est la méthode recommandée pour écouter les événements. Elle permet d'attacher une fonction à un événement spécifique. 

// Syntaxe générale
element.addEventListener('typeEvenement', function() {
    // Code à exécuter
});

// Exemple concret
document.querySelector('.my-button').addEventListener('click', function() {
    console.log('Bouton cliqué !');
});

Types d'événements courants

Événements de souris

// Clic simple
element.addEventListener('click', function() { });

// Survol
element.addEventListener('mouseenter', function() { });

// Sortie de survol
element.addEventListener('mouseleave', function() { });

// Mouvement de souris
element.addEventListener('mousemove', function() { });

Événements de clavier

// Touche enfoncée
document.addEventListener('keydown', function() { });

// Touche relâchée
document.addEventListener('keyup', function() { });

// Détection d'une touche spécifique
document.addEventListener('keydown', function(event) {
    if (event.key === 'g') {
        console.log('Touche G pressée !');
    }
});

Événements de formulaire

// Focus sur un champ
input.addEventListener('focus', function() { });

// Perte de focus
input.addEventListener('blur', function() { });

// Changement de valeur
input.addEventListener('change', function() { });

// Saisie en temps réel
input.addEventListener('input', function() { });

Événements de fenêtre

// Redimensionnement de fenêtre
window.addEventListener('resize', function() { });

// Défilement de page
window.addEventListener('scroll', function() { });

// Chargement de page
window.addEventListener('load', function() { });

Consigne

Ajouter la classe .highlight sur la carte concernée en fonction de l'événement indiqué

Exemple


          const myElement = document.querySelector('.my-element');
          const btn = document.querySelector('.my-element .btn');

          btn.addEventListener("click", function() {
            myElement.classList.add("highlight");
          })
